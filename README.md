
🏥 MedR – Patient Record Management System

A modern Patient Record Management System built using HTML5, CSS3, and Vanilla JavaScript.

MedR is a single-page web application that allows healthcare staff to manage patient records efficiently using the browser's Local Storage without requiring a backend server.
